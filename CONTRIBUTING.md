# Contributing to Wanderlust Adventure

Thank you for your interest in contributing to **Wanderlust Adventure**.  
This is a frontend-only project built using **HTML and CSS**, and all contributions are welcome.

Live demo: https://wanderlust-adventure-by-sohaib.vercel.app

---

##  How You Can Help

You can contribute by:
- Improving UI/UX and layouts
- Fixing bugs or styling issues
- Making the design more responsive
- Improving accessibility
- Updating documentation
- Suggesting new sections or components
